Section 1: Front-End Web Development

How Does the Internet Actually Work?

If I want to access Google, first:
1. Client's browser sends request to Internet Service Provider (ISP).
2. ISP sends same request to DNS (Domain-name System) sever.
3. DNS searches in it's database the IP address of Google website.
4. DNS sends IP address back to ISP and then to client's browser.
5. Client's browser sends another request to ISP via the Internet backbone to the server located at the IP address, in this case the Google server.

How Do Websites Actually Work?

When inspecting a webpage, if you modify something and then refresh, this change will be lost because the change is only made on a local copy from the server. When clicking refresh, we create a new request to the server which sends a fresh copy of the website.

HTML is responsible for the structure of the website, I.e. add images, textbox or buttons.
CSS is responsible for styling your website. I.e. make button red and change font and size.
JS is responsible for adding interaction and behaviors. Give behavior to button.
